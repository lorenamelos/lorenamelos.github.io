About

Engineer, not just scientist

I started as a Data Scientist — building forecasting models, NLP pipelines, and ML systems. But I kept noticing the same pattern: models that worked in notebooks failed in production.

That frustration led me to AI Engineering: the discipline of building intelligent systems that actually survive contact with real users, real data, and real infrastructure.

Today I focus on the full journey — from messy data to deployed product. Every system I build follows what I call defensive architecture: code that fails gracefully, recovers automatically, and produces auditable results.

"Write code that survives, not merely functions."

— AIRA Labs technical philosophy
